# Astoria CPW Concierge — Windows Desktop App

## Quick Start (run without building)
1. Install Node.js from https://nodejs.org (LTS version)
2. Open a terminal in this folder
3. Run: npm install
4. Run: npm start

## Build Windows .exe Installer
1. Install Node.js from https://nodejs.org
2. Open terminal in this folder
3. Run: npm install
4. Run: npm run build-win
5. Find the installer in the /dist folder: "Astoria CPW Concierge Setup 1.0.0.exe"
6. Run the installer on any Windows PC — creates a Start Menu shortcut and desktop icon
